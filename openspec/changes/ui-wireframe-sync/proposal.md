## Why

초기 구현 시 스토리보드 v2 와이어프레임의 UI 레이아웃 명세가 스펙에 포함되지 않아, 코드와 스펙이 불일치했다. 기능 동작(API, 데이터 모델)은 스펙에 정의되어 있으나, 화면 레이아웃·컬러·컴포넌트 스타일 요구사항이 누락된 상태다. 이를 스펙에 반영해 향후 AI 구현 시 와이어프레임과 일관된 UI가 생성되도록 한다.

## What Changes

- **로고 컴포넌트**: 모든 화면에서 teal 배경 박스 안에 흰색 텍스트 `TaskFlow`로 표시
- **헤더 탭 내비게이션**: 칸반/채팅/멤버 탭이 헤더 중앙에 위치, 현재 페이지 탭은 teal 채움 버튼
- **칸반 컬럼 색상**: TODO=amber, DOING=blue, DONE=green 배경으로 상태를 시각적으로 구분
- **팀 선택 레이아웃**: 팀 만들기와 초대코드 합류 영역이 가로 2컬럼으로 나란히 배치
- **인라인 태스크 입력**: 카드 생성 시 제목 입력과 함께 담당자 드롭다운(팀 멤버 목록)이 표시
